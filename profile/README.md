# Macbot Open Source (MOS) 🌿

"Macbot" tecrübesini temel alan, minimalist, açık kaynaklı ve kullanıcı odaklı bir uygulama ekosistemi.

## 🚀 Vizyonumuz
MOS, karmaşadan uzak, reklamsız ve özgür yazılım prensiplerine sadık kalarak günlük dijital ihtiyaçlar için hafif araçlar geliştirmeyi amaçlayan bir projedir.

## 🛠️ Neler Planlıyoruz?
Şu anda altyapı çalışmalarımız devam ediyor. Yakında burada yerini alacak olan projelerimiz:
- **Macbot:** Topluluk yönetimi için geliştirilmiş açık kaynaklı Discord asistanı.
- **MOS Apps:** Saat, hava durumu ve günlük araçlardan oluşan hafif uygulama serisi.
